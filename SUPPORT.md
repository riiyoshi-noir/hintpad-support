# HintPad Support

## About

HintPad is a learning-focused note-taking app for macOS with highlighting, hint notes, and sub-pages.

## System Requirements

- macOS 14.0 (Sonoma) or later

## Features

- **Highlighter & Underline** - 7 colors to mark important text
- **Hint Notes** - Register meanings for key terms, hover to review
- **Sub-pages** - Create linked sub-pages from selected text
- **Multi-pane** - View multiple notes side by side
- **Heading sizes** - Organize notes with H1, H2, H3 headings
- **Pin notes** - Keep frequently used notes at the top

## FAQ

**Q: Where is my data stored?**
A: All data is stored locally on your Mac. HintPad does not use cloud storage.

**Q: Can I export my notes?**
A: This feature is planned for a future update.
